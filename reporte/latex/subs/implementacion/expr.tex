Este módulo se encarga de definir el tipo de dato
para construir las expresiones de nuestro lenguaje,
el cálculo SKI, y de permitir su
representación como una cadena en forma de árbol binario.

Así, se definen 5 constructores para los distintos tipos
de expresión:
\begin{itemize}
\item \boldit{SComb}: Construye el operador $S$.
\item \boldit{KComb}: Construye el operador $K$.
\item \boldit{IComb}: Construye el operador $I$.
\item \boldit{Str}: Construye una cadena.
\item \boldit{EApp}: Construye una aplicación de función en la
  que la primera expresión representa la función a aplicar y
  la segunda el argumento con el que se va a realizar la aplicación.
\end{itemize}
